Introduction

Overview

BIGSS xreg GUI is a user-friendly interface designed to simplify complex medical imaging tasks such as Digital Reconstructed Radiograph (DRR) generation and image registration. It leverages the capabilities of the xreg library, making advanced image processing accessible to users without extensive programming knowledge.

The GUI guides users through selecting processes, configuring parameters, and executing tasks, providing visual feedback and process monitoring.

The primary purpose of the BIGSS xreg GUI is to provide an intuitive interface for 2D/3D registration tasks. Tge GUI simplifies the workflow for generating DRRs and performing image registration by utilizing automatically generated config scripts to handle xreg execution. It also offers visualizations of registration results and intermediate outputs.

For the original xreg library, please visit the following Github repository here: https://github.com/rg2/xreg

For the current BIGSS version of the xreg library, please visit: https://github.com/jhu-bigss/xreg

Some Features

  • Process Selection: Easily select from available processes such as DRR generation and various registration workflows.

  • Configuration Management: Create, edit, and save configuration files through an interactive interface.

  • Process Execution: Run selected processes with real-time output display.

  • Visualization: Visualize registration scenes and DRR outputs within the GUI.

  • Customization: Set up and adjust parameters specific to each process.
